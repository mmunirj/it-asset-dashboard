**Overview**

This project analyses an internal IT asset inventory dataset to uncover insights about device usage, age, and distribution across company locations.

**Objectives**

Clean and transform asset data using pandas

Calculate device age and identify outdated hardware

Visualize asset distribution and health with Seaborn and Matplotlib

**Key Insights**

London HQ has the most assets deployed

Dell Latitude 5530 is the most common model

Average device age: 3.1 years

Around 20% of assets are near end-of-life

**Tools Used**

Python (pandas, numpy, seaborn, matplotlib)

Jupyter Notebook / Spyder

CSV data source (mock inventory)
